## Folder and File Details

- **`README.md`** (Authoritative): This file. A guide to navigating the Antigravity folder and understanding my contributions.
- **`Summary of Only Necessary Context.md`** (Authoritative, Ephemeral): A rewritten-each-session summary that contains only what is essential for resuming work in the next session. It drops all old or completed paths to keep context clean.
- **`Session Summaries/`** (Authoritative): Contains human-readable summary reports generated at the end of each session (`HumanReport<N>.md`). These serve as a historical record of what was accomplished, challenges overcome, decisions made, and next steps planned.

## Navigation Guide

If you are entering this folder without prior context:
1. **To understand what I did last or historically:** Read the latest files in `Session Summaries/`.
2. **To understand what I am currently working on or about to do:** Read `Summary of Only Necessary Context.md`.
3. **To see my communications:** Track the `/chats/` directories listed above.
